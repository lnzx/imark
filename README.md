## imark 

chrome 书签插件

#### 构建工具

`nodejs`, `vitejs`, `yarn`

#### 依赖

`bulma`css

#### 开始使用

1. 直接下载release里面打好的包，安装即可。

2. 也可以自己编译：

克隆本仓库，进入项目文件夹，运行以下命令：

`yarn` 

`yarn build` 

`dist`目录下会生成`imark`文件夹即插件

#### 功能

* 书签列表

* 拖拽删除书签

#### 缺陷

只支持1级自定义目录，即 `书签栏` > `自定义目录` > `书签`

#### future

* 支持多级目录
* 拖动书签
* 批量打开
